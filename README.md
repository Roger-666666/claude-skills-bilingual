## 🛠️ Skills List

### 1️⃣ Prompt Deep Creator
<details>
<summary><b>Click to expand: Transform vague ideas into professional Prompts 🚀</b></summary>

#### 🔹 Core Function
Transform scattered, colloquial, or even vague creative ideas into high-quality, well-structured, ready-to-use formal prompts.
- **Element Extraction**: Automatically identify core objectives, audience, style, and constraints.
- **Structured Output**: Generate standard templates including role settings, task background, core tasks, and prohibitions.
- **Smart Follow-up**: Actively guide supplementary key elements when input information is insufficient.

#### 🔹 Use Cases
- **Writing**: Technical blogs, in-depth articles, popular science articles.
- **Planning**: Business proposals, Product Requirements Documents (PRD).

#### 🔹 Installation
1. Download `prompt-deep-creator.zip` and extract it.
2. Copy the `skill.md` content to your Claude preset instructions, or place the extracted folder in the software's `skills` directory.
3. If using `skills.json` management, add the corresponding path.
</details>

### 2️⃣ Three Provinces and Six Ministries (Sansheng)
<details>
<summary><b>Click to expand: Universal Task Processing Administrative System ⚔️</b></summary>

#### 🔹 Core Function
Structure any complex task according to the strict process of "Sorting → Planning → Review → Dispatch → Execution → Summary".
- **Crown Prince · Task Sorting**: Determine task type and extract core requirements.
- **Secretariat · Draft Plan**: Formulate specific, executable plans (reject empty talk).
- **Chancellery · Review & Gatekeeping**: **Core Feature**, substandard plans will be directly "rejected".
- **Six Ministries · Actual Execution**: Ministry of War (Technology), Ministry of Rites (Documents), Ministry of Revenue (Data) and other departments produce actual results.

#### 🔹 Installation
1. Download `The Three Provinces and Six Ministries System AI Agent (win)_skill_v1.0.zip` and extract it.
2. **CLI Mode**:
   - Ensure Python 3 is installed.
   - Run `python3 edict.py '{"task":"task content"}'` in the terminal to trigger the workflow.
3. **Web Kanban Mode**:
   - Run `python3 scripts/serve.py` to start the visual interface (visit 127.0.0.1:7891).
</details>

## 📥 General Installation & Configuration

For mainstream Claude terminal tools, you can integrate as follows:

### Method A: Directory Integration (Recommended)
1. Find your software installation directory (e.g., `~/.gemini/antigravity/scratch/`).
2. Copy the corresponding skill folder to the `skills/` directory.
3. Restart Claude CLI, try typing "Use Sansheng to process..." or "Help me deeply create a prompt".

### Method B: Configuration File Association
Add the following configuration to your `skills.json`:
```json
{
  "skills": [
    {
      "name": "sansheng",
      "path": "./skills/sansheng/SKILL.md"
    },
    {
      "name": "prompt-deep-creator",
      "path": "./skills/prompt-deep-creator/skill.md"
    }
  ]
}
```
