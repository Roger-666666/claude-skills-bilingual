# Claude Code Skills - Installation Guide

## Quick Start

### 1. Download

Download the bilingual package:
- `claude-skills-bilingual.tar.gz` - Complete bilingual package

### 2. Extract

```bash
tar -xzf claude-skills-bilingual.tar.gz
cd claude-skills-bilingual
```

### 3. Choose Language

```bash
# Switch to English (default)
./switch_lang.sh en

# Switch to Chinese
./switch_lang.sh zh
```

## Integration with Claude CLI

### Method 1: Copy to Skills Directory

```bash
# Find your Claude CLI skills directory
# Example paths:
# - ~/.gemini/antigravity/scratch/skills/
# - ~/.claude/skills/
# - Check your Claude CLI documentation

# Copy skills
cp -r skills/sansheng ~/.gemini/antigravity/scratch/skills/
cp -r skills/prompt-deep-creator ~/.gemini/antigravity/scratch/skills/
```

### Method 2: Configure skills.json

Add to your `skills.json`:

```json
{
  "skills": [
    {
      "name": "sansheng",
      "path": "./skills/sansheng/SKILL.md"
    },
    {
      "name": "prompt-deep-creator",
      "path": "./skills/prompt-deep-creator/skill.md"
    }
  ]
}
```

## Troubleshooting

### Issue: Python not found

Make sure Python 3 is installed:
```bash
python3 --version
# or
python --version
```

### Issue: Claude CLI not detected

The skill will work in "framework mode" without Claude CLI. To enable full LLM mode:
```bash
# Install Claude CLI first, then:
claude --model  # Select an available model
```

### Issue: Permission denied

Make scripts executable:
```bash
chmod +x switch_lang.sh
chmod +x scripts/*.py
```
